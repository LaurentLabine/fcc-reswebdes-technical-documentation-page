What is discourse

Discourse is an open source forum made available to host your very own online community. It is made available for your personal use free of charge and come with the promise of always being free. Here is how to get your very own forum running hosted locally on your windows machine.

Installation Requirements

Discourse is made available to windows user through the Windows Subsystem for Linux

Install WSL2

The setup requires the WSL 2 installation. It is only available in Windows 10 builds 18917 or higher. Open the command menu and use the winver command to validate the version currently installed. If your version is lower, Update to the latest one before proceeding. Once this has been validated, proceed with the following :

  1. Enable the Windows Subsystem for Linux

    You must first enable the "Windows Subsystem for Linux" optional feature before installing any Linux distributions on Windows. Open PowerShell as Administrator and run:

    dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

    In order to update to WSL 2, restart your machine and follow these steps:.

  2. Check requirements for running WSL 2

    Updating to WSL 2 requires you to be running Windows 10 build 18362 or higher.

  3. Enable Virtual Machine feature

    Virutal machine capabilities are required in order to utilize this feature. In order to enable them open a powershell prompt and run :

    dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

  4. Download the Linux kernel update package

    Download the latest version from Here and install it on your machine using elevated permissions

  5. Set WSL 2 as your default version

    Use wsl --set-default-version 2 to set WSL2 as your default version

  6. Install your Linux distribution of choice

    Make your way to the Microsoft store and search and download your favorite Linux distribution

    The following links will open the Microsoft store page for each distribution:

    • Ubuntu 16.04 LTS
    • Ubuntu 18.04 LTS
    • Ubuntu 20.04 LTS
    • openSUSE Leap 15.1
    • SUSE Linux Enterprise Server 12 SP5
    • SUSE Linux Enterprise Server 15 SP1
    • Kali Linux
    • Debian GNU/Linux
    • Fedora Remix for WSL
    • Pengwin
    • Pengwin Enterprise
    • Alpine WSL


    Upon launching the machine for the first time you will be asked to create a username and password

Install Discourse Dependencies

From your WSL terminal run the following:

bash <(wget -qO- https://raw.githubusercontent.com/techAPJ/install-rails/master/linux)

which will install the following :

  • Git 113
  • rbenv
  • ruby-build
  • Ruby
  • Rails
  • PostgreSQL
  • SQLite
  • Redis
  • Bundler
  • MailCatcher
  • ImageMagick
Clone Discourse

From a terminal, run the following :

mkdir discourse
cd discourse

Which will make a discourse folder and position yourself in it. Once in use

git clone https://github.com/discourse/discourse.git ~/discourse

To clone the discourse project from either the official repository or a forked version from yours.

Setup Database

Enter the following to create a role with the same name matching your operating system's username

sudo -u postgres createuser -s "$USER"

Bundle Discourse

Now position yourself from within your discourse folder and install the gems by using :

source ~/.bashrc
bundle install

Now Bundle your installation by using :

bundle exec rake db:create
bundle exec rake db:migrate
RAILS_ENV=test bundle exec rake db:create db:migrate

And you are ready to dun the specs

bundle exec rake autospec

And start the server by using

bundle exec rails server

And that is IT folks! You should have your own server available for hours of fun should you have to debug some styling discrepancies!
