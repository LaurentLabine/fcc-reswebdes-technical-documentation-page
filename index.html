What is discourse

Discourse is a from-scratch reboot, an attempt to reimagine what a modern Internet discussion forum should be today, in a world of ubiquitous smartphones, tablets, Facebook, and Twitter. It supports the latest, stable releases of all major browsers and platforms and this guide will guide you through setting up and hosting your very own discourse forum locally on your machine.

Requirements

WSL2

The setup requires the WSL 2 installation. It is only available in Windows 10 builds 18917 or higher. Here are the steps on how to set it up:

  1. Enable the Windows Subsystem for Linux

    You must first enable the "Windows Subsystem for Linux" optional feature before installing any Linux distributions on Windows. Open PowerShell as Administrator and run:

    dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

    In order to update to WSL 2, restart your machine and follow these steps:.

  2. Check requirements for running WSL 2

    Updating to WSL 2 requires you to be running Windows 10 build 18362 or higher.

Check your build version using winver command from the run command menu.

If your version is lower, Update to the latest one before proceeding.
